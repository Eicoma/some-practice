MG动画相关脚本及应用

作者：Eicoma 2021-05-07

MG动画相关脚本及应用

  1. 【线条】【由线条构成的简单图形】类的动画，可用【描边】添加动态 改变（画笔大小）和（开始结束）
  2. 【最大化面板】在合成一栏右键选择
  3. 调整各图形的【关键帧位置】【素材本身出现位置】来调整效果
  4. 选中要隐藏的素材，勾选【副开关】再勾选【总开关】

文字 效果案例一

  1. 画一条斜线的形状图层，添加中继器，改变位置使斜线更紧密， 增加副本以覆盖横向的文字，再添加一个中继器，增加副本，调整位置 以覆盖纵向的文字-把形状图层放置在文字下方-选择alpha遮罩
  2. 【线性擦除】效果
  3. 【制作边缘】将文字或图形单独复制一层-在该层添加【描边】调整画笔大小，开始结束
  4. 给文字绘制【蒙版】（简单绘制，不必描边）-添加【描边】-选择所有蒙版 -调整颜色-画笔大小设置到和文字粗细一致-调整开始结束
  5. 按住shift+ctrl可从中心画出正圆图形
  6. 给形状图层添加【修剪路径】-调整开始，结束，偏移
  7. 右键-时间-冻结帧（用于动画中的停顿，某一画面的保留）

LOGO 效果案例一

  1. 用钢笔绘制一个形状图层（如波浪形）-设置填充色-位置关键帧（如从下往上经过）
  2. 将形状图层放在LOGO上边，在LOGO层选择alpha遮罩
  3. 在效果预设里-【Text】-【3D Test】内有多种动态效果，可调节图层上的关键帧
